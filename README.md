# RiskAI — Система интеллектуального анализа финансовых рисков

**Сайт:** [https://riskai.ru](https://riskai.ru)

## RISKAI

RiskAI — это клиент-серверная система для анализа финансовых рисков с использованием методов искусственного интеллекта и машинного обучения. Проект демонстрирует применение ИИ для оценки кредитного, инвестиционного и страхового риска, включая выявление аномалий, прогнозирование дефолта и анализ устойчивости портфелей.

## Стэк

### **Frontend:**

* React — интерфейс визуализации результатов анализа.

### **Backend:**

* Python — API, обработка данных, интеграция с ML.
* FastAPI / Flask — серверная логика.

### **ML:**

* Машинное обучение для анализа финансового риска.
* Отдельный ML‑сервис для расчётов.

### **Инфраструктура:**

* Docker / Docker Compose — контейнеризация всех компонентов.
* Nginx — маршрутизация и проксирование запросов.
* Поддержка переменных окружения и Yandex Object Storage для датасетов.
